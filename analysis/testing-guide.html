Features

Image Input Methods

  • Drag & drop file upload
  • Click to select photo from device
  • Real-time webcam capture
  • Image preview before analysis
  • Support for JPEG and PNG formats
  • Max file size 10MB with validation

Analysis Results

  • Face shape detection (7 types)
  • Confidence percentage (0-100%)
  • Optional facial measurements
  • 3-5 personalized hairstyle recommendations
  • Match scores for each recommendation
  • Detailed hairstyle information

User Experience

  • Loading state with progress animation
  • Detailed error messages with suggestions
  • Mobile-responsive design (all devices)
  • Fast performance (<5s typical)
  • Share results functionality
  • Privacy notice displayed prominently

Error Handling

  • No face detected - helpful message
  • Poor image quality - suggestion to improve
  • File too large - size requirement shown
  • Network errors - automatic retry logic
  • API errors - user-friendly explanations
  • Browser compatibility checks

API Documentation

Endpoint

POST /api/v1/analyze-and-recommend

Request Format

Content-Type: multipart/form-data

Field: image (JPEG or PNG file)

Response Format

{ "face_shape": "oval", "confidence": 0.95, "measurements": { "width": 145.5, "height": 195.3, "forehead_width": 130.2, "jaw_width": 125.8 }, "recommendations": [ { "name": "Layered Bob", "description": "Classic layered bob", "match_score": 0.92, "image_url": "https://...", "tags": ["Trendy", "Easy"] } ] }

Error Response Examples

No Face Detected (400):

{ "error": "No face detected", "message": "Could not detect a face..." }

Server Error (500):

{ "error": "Internal server error", "message": "An unexpected error occurred..." }

Common Issues

Webcam not working?

  • Check HTTPS (required in production)
  • Verify browser permissions
  • Try different browser

API not responding?

  • Check endpoint URL in analysis.js
  • Verify backend is running
  • Check CORS headers
